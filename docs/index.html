Tilt Shift Camera

A shader that blurs parts of the scene where depth is close to the near or far plane, creating a tilt-shift camera like effect that works best in games with far away sceneries. Allows User edits of the blur ranage and strength through reshade GUI.

Additional Details

Additional Scene Light

A shader that adds a user-defined lightsource that would add an additional layer of lighting to all world objects as if they had Phong texture. Allows user edits to change properties of the light as well as the phong material through ReShade GUI.

Ambient Occlusion

A shader that makes use of the depth buffer and a predefined fractal texture to render a fog-like effect in game scene. Fog becomes denser the greater a pixel's depth is. Allows user edits to change the depth at which backbuffer completely fades.

God Ray

A shader that creates a god ray effect that would cast rays from bright scene objects by repeatedly sampling pixel towards a user defined direction. Depth is used as a multplicative factor in the sampling strength to capture decay over distance from the camera as well as occulsion. Allows user edits to ray cast direction, sample rates, and ray color.
